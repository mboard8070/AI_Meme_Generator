## Features

- **AI-Driven Meme Generation**: Leverages artificial intelligence to create humorous, context-aware memes.
- **Automated Posting**: Built-in support for auto-posting to Instagram (via session management) and extensible to other platforms.
- **Modular Architecture**: Clean separation of configuration, utilities, and workflows for easy customization and maintenance.
- **Lightweight & Extensible**: Pure Python implementation with dependencies managed via `requirements.txt`.
